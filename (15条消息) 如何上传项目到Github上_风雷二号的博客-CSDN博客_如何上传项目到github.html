如何上传项目到Github上

        注：

 

此文章受众群体为上传过项目但是相隔很长时间以后又上传然后忘了步骤的人（比如我）。

1. 找到装项目的文件夹，右键点击 Git Bash Here

                                ​​​​​​​        

2.输入指令：     git init   

3.输入指令        git add .   （git空格add空格.)

4.输入指令       git commit -m "上传"    （双引号部分是上传注释，可为空）

        如果显示  error: remote origin already exists. 错误信息，

        输入 git remote rm origin，再执行4.输入指令

               

5. 找到GitHub上找到新建的存储库的地址

                        

6.输入指令：git remote add origin 存储库地址

7.输入指令：git push -u origin master 

不出意外就上传成功了。出了意外就多试几次。

（今天在网上找上传教程，找了很多都是要输入git安装时设置的账号密码，我这种二十年上传一次的，连上传步骤都忘了的人，怎么会记得这些东西，后来看了看发现传过项目的都可以不用账号密码就行了。）

写下这篇文章，防止以后再忘了步骤耗费时间去查找。
